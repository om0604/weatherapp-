# Weather Agent

A smart weather assistant built with Next.js, tailored for Indian cities.

## Deployed Version
[Live Demo](https://weatherapp-5nnhykwjz-oms-projects-439a14d6.vercel.app/)

## Getting Started

1. **Clone the repository** (if you haven't already).

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the app**:
   Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Icons)
- [Mastra](https://mastra.ai/) (Agent Backend)
